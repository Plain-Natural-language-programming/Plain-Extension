# Plain Language Theme + Icons

Candy color theme and custom icons for Plain Language (.pln) files.

## Features
- Plain Language Candy color theme
- Plain Language syntax highlighting
- Plain Language language icon
- Optional icon themes (Plain Language Icons, Seti + Plain)

## Enable
1. Open the Command Palette.
2. Run "Preferences: Color Theme" and select "Plain Language Candy".
3. Run "Preferences: File Icon Theme" and select either "Plain Language Icons" or "Seti (VS Code) + Plain" to apply the .pln icon.
